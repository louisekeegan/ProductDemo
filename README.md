# ProductsDemo

A full-stack **ASP.NET Core (.NET 9)** sample application demonstrating:

- FastEndpoints-based Web API  
- PostgreSQL database  
- Dapper data access  
- Integration tests using Testcontainers  
- Blazor UI (Interactive Server)  
- .NET Aspire orchestration
